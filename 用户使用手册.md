## 1. 系统简介

本系统是 **The Infinite Cafe（无限咖啡馆）** 的仓储管理系统（WMS），用于管理咖啡馆的原料、物料的入库、出库、移库、盘库等日常仓储操作。

**核心功能：**

| 功能模块 | 说明 |
|---------|------|
| 首页概览 | 展示物料总数、仓库数量、入库单/出库单等关键指标，以及库存分布图表和进出货趋势 |
| 库存统计 | 按仓库或原料维度查看当前库存情况 |
| 库存记录 | 查看所有库存变动的历史记录 |
| 入库 | 创建入库单，记录原料进货入库 |
| 出库 | 创建出库单，记录原料出库 |
| 移库 | 将原料从一个仓库转移到另一个仓库 |
| 盘库 | 定期清点库存，核对账面库存与实际库存 |
| 基础数据 | 管理仓库、往来单位、品牌、商品等基础信息 |

## 2. 登录与退出

### 2.1 登录

1. 打开浏览器，访问系统地址
2. 在登录页面输入 **账号** 和 **密码**
3. 输入图形 **验证码**
4. 可勾选「记住密码」，下次登录时自动填充
5. 点击「**登 录**」按钮

> 💡 **提示：** 如果验证码看不清，可以点击验证码图片刷新。

### 2.2 退出

点击页面右上角的用户头像，选择「**退出登录**」即可安全退出系统。

### 2.3 修改密码

点击页面右上角的用户头像 → 「**个人中心**」 → 「**重置密码**」标签页，输入旧密码和新密码后提交即可。

## 3. 首页概览

登录成功后进入首页，首页展示以下内容：

### 3.1 统计卡片

页面顶部展示四个统计卡片：

### 3.4 各分类进出货金额对比

以柱状图展示各分类原料的进货金额与出货金额对比。

### 3.5 近14天进出货趋势

以折线图展示最近14天每日的进货金额与出货金额趋势变化。

## 4. 库存统计

**菜单路径：** 侧边栏 → 「库存统计」

库存统计页面用于查看当前系统内所有原料的库存情况。

### 4.1 查询维度切换

页面顶部可以切换两种查询维度：
- **仓库维度**：按仓库 → 原料 → 规格的层级展示库存
- **原料维度**：按原料 → 规格 → 仓库的层级展示库存

### 4.2 搜索过滤

支持以下搜索条件：
- 仓库（下拉选择）
- 原料名称
- 原料编号
- 规格名称
- 规格编号

### 4.3 过滤零库存

勾选「**过滤掉库存为0的原料**」可以隐藏已清零的原料，只展示有库存的原料。

**菜单路径：** 侧边栏 → 「库存记录」

## 6. 入库管理

**菜单路径：** 侧边栏 → 「入库」

入库管理用于记录原料进货入库操作。

### 6.1 入库单列表

进入入库页面后，可以看到所有入库单的列表，支持按单号、状态等条件搜索。

### 6.2 新建入库单

1. 点击「**新增**」按钮，进入入库单编辑页面
2. 系统自动生成入库单号（格式：`RK + 时间戳`）
3. 填写入库单基本信息：

| 字段 | 必填 | 说明 |
|------|------|------|
| 入库单号 | ✅ | 系统自动生成，也可手动修改 |
| 仓库 | ✅ | 选择入库的目标仓库 |
| 入库类型 | 否 | 选择入库类型 |
| 往来单位 | 否 | 选择供应商 |
| 业务单号 | 否 | 关联的外部业务单号 |
| 备注 | 否 | 备注信息 |

4. 点击「**添加原料**」按钮，在弹出的原料选择框中勾选需要入库的原料
5. 在明细列表中填写每项原料的 **数量** 和 **金额**
6. 总数量和总金额会自动计算

### 6.3 操作按钮

| 按钮 | 说明 |
|------|------|
| **暂存** | 保存入库单但不执行入库，状态为"暂存"，可稍后继续编辑 |
| **完成入库** | 确认入库，库存会实际增加，入库单状态变为"已完成" |
| **作废** | 将已保存的入库单作废（仅限未完成的单据） |
| **取消** | 放弃编辑，返回入库单列表 |

### 6.4 编辑已有入库单

在入库单列表中，点击操作列的「**编辑**」可修改暂存状态的入库单。

## 7. 出库管理

**菜单路径：** 侧边栏 → 「出库」

出库管理用于记录原料出库操作。流程与入库类似。

### 7.1 新建出库单

1. 点击「**新增**」按钮，进入出库单编辑页面
2. 系统自动生成出库单号（格式：`CK + 时间戳`）
3. 填写出库单基本信息：

| 字段 | 必填 | 说明 |
|------|------|------|
| 出库单号 | ✅ | 系统自动生成 |
| 仓库 | ✅ | 选择出库的源仓库 |
| 出库类型 | ✅ | 选择出库类型 |
| 往来单位 | 否 | 选择客户/接收方 |
| 业务单号 | 否 | 关联的外部业务单号 |
| 备注 | 否 | 备注信息 |

4. 点击「**添加原料**」，从当前仓库的现有库存中选择需要出库的原料
5. 填写出库 **数量** 和 **金额**

> ⚠️ **注意：** 出库时只能选择所选仓库中有库存的原料，且出库数量不能超过当前库存量。

### 7.2 操作按钮

| 按钮 | 说明 |
|------|------|
| **暂存** | 保存出库单但不执行出库 |
| **完成出库** | 确认出库，库存会实际减少 |
| **作废** | 将出库单作废 |
| **取消** | 放弃编辑 |

## 8. 移库管理

**菜单路径：** 侧边栏 → 「移库」

移库功能用于将原料从一个仓库转移到另一个仓库。

### 8.1 新建移库单

1. 点击「**新增**」按钮
2. 系统自动生成移库单号（格式：`YK + 时间戳`）
3. 填写移库单基本信息：

| 字段 | 必填 | 说明 |
|------|------|------|
| 移库单号 | ✅ | 系统自动生成 |
| 源仓库 | ✅ | 原料当前所在的仓库 |
| 目标仓库 | ✅ | 原料要转移到的仓库 |
| 备注 | 否 | 备注信息 |

4. 点击「**添加原料**」，从源仓库库存中选择需要移库的原料
5. 填写移库 **数量**

### 8.2 操作按钮

| 按钮 | 说明 |
|------|------|
| **暂存** | 保存移库单但不执行 |
| **完成移库** | 确认移库，源仓库库存减少，目标仓库库存增加 |
| **作废** | 将移库单作废 |
| **取消** | 放弃编辑 |

## 9. 盘库管理

**菜单路径：** 侧边栏 → 「盘库」

盘库功能用于定期清点实际库存，核对账面库存是否一致。

### 9.1 新建盘库单

1. 点击「**新增**」按钮
2. 在页面中选择需要盘点的 **仓库**
3. 点击「**开始盘库**」
4. 系统会自动加载该仓库下所有有库存的原料，显示 **账面库存**
5. 逐项填写 **实际库存** 数量
6. 系统自动计算 **盈亏数**（实际库存 - 账面库存）
7. 如果发现仓库中有新的原料（账面上没有），可以点击「**新增库存**」添加

### 9.2 盈亏说明

| 盈亏数 | 含义 |
|--------|------|
| 正数（+） | 盘盈：实际库存多于账面库存 |
| 0 | 账实相符 |
| 负数（-） | 盘亏：实际库存少于账面库存 |

### 9.3 操作按钮

| 按钮 | 说明 |
|------|------|
| **暂存** | 保存盘库单但不生效，可稍后继续盘点 |
| **完成盘库** | 确认盘库结果，系统会按实际库存更新账面库存 |
| **作废** | 将盘库单作废 |
| **取消** | 放弃编辑 |

## 10. 基础数据管理

**菜单路径：** 侧边栏 → 「基础数据」

基础数据模块用于维护系统核心主数据。

### 10.1 仓库管理

管理系统中的仓库信息。

**操作说明：**

- **新增**：点击「新增」，填写单位信息

| 手机号 | 联系电话 |

管理原料/商品的品牌信息。

**操作说明：**
- **新增品牌**：点击「新增」，输入品牌名称
- **修改/删除**：在列表中操作

**商品信息：**

| 字段 | 说明 |
|------|------|

## 12. 常见问题

### Q1: 验证码看不清怎么办？
点击验证码图片即可刷新，获取新的验证码。

### Q2: 入库时找不到想要的原料？
请先在「基础数据 → 商品管理」中添加该原料和对应的规格（SKU），然后再创建入库单。

### Q3: 出库时提示库存不足？
出库数量不能超过该仓库中该原料的当前库存量。请在「库存统计」中确认库存情况。

### Q4: 移库时源仓库和目标仓库有什么要求？
源仓库和目标仓库不能相同。源仓库中必须有需要移库的原料库存。

### Q5: 盘库时发现新原料怎么办？
在盘库页面点击「新增库存」按钮，选择新发现的原料并填写实际库存数量。

### Q6: 暂存的单据可以修改吗？
可以。暂存状态的单据（入库单/出库单/移库单/盘库单）都可以重新编辑。

### Q7: 已完成的单据可以撤销吗？
已完成的单据不可撤销。如果需要调整，请创建新的反向操作单据（如：误入库后创建出库单）。

### Q8: 作废的单据会影响库存吗？
作废操作只适用于尚未完成入库/出库/移库/盘库的暂存单据，不会影响实际库存。
